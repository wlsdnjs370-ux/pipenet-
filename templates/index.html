      SDF/CAD 구조 분석 보조 리포트

      동일 SDF와 선택 CAD 파일을 기준으로 최원단 헤드 30개, 배관 길이·구경·분기 변화 대조와 CAD 자동 인식 후보를 함께 확인합니다.

      DXF/SDF 뷰어에서 파일을 업로드하면 보조 리포트가 자동 생성됩니다.

      Remote 30 추출 모듈

      DXF에서 배관/헤드/관경 문자를 추출하고, 알람밸브 기준 최원단 30개 헤드를 선정해 Isometric PNG와 PIPENET 호환 Excel을 생성합니다.

      1. DXF 파일

      한국 도면의 sprinkler 레이어 패턴(-소화(SP가지관), -소화(SP헤드하향) 등)에 맞춰 자동 인식합니다.

      2. 알람밸브 위치

      자동 검출은 블록명/레이어/TEXT에서 ALARM/ALV 키워드를 탐색합니다. 실패 시 헤드 군집 centroid로 fallback.

      3. Remote 선정 기준

      Hydraulic 모드: 선정된 N개만 동시방수 가정 → Hazen-Williams (C=120) 마찰손실 + 정수두 합산. 가장 큰 손실 헤드 = 진짜 hydraulic remote.

      4. Hydraulic 파라미터

      5. 출력 옵션

      PIPENET 호환 Excel은 항상 생성됩니다. SDF는 RV03_NEW.sdf와 동일 스키마.

      6. 고급 설정

      레이어 키워드 / 허용오차 / 스케일
