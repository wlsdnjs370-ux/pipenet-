Remote 30 프로토타입 — 통합

평면도 추출과 계통도 추출 결과를 결합하여 펌프 → 밸브 → 가지배관 → 30 헤드 전 구간의 연산 가능한 SDF 를 생성합니다.

입력 준비 상태
평면도 결과: 미준비
계통도 결과: 미준비
기계실 결과: 미준비 (선택)

[평면도 추출] 과 [계통도 추출] 은 필수, [기계실 추출] 은 선택입니다. 기계실을 추출하면 옥상수조가 새 경계 Input(대기압)으로 라이저 상단에 연결됩니다.

결합 방식
가압송수방식 (통합 전 선택)

화재안전기준(NFPC 103) 3점 성능곡선으로 자동 구성됩니다. 정격값만 조정하세요.

수원이 최저 헤드보다 몇 m 아래(B1 등)인지. 이 고저차가 펌프 실양정에 더해져 계산 결과를 바꿉니다. 도면에 높이 정보가 없어 직접 입력해야 정확합니다.

체절양정 = 정격×1.40, 150% 유량점 = 정격유량×1.50 / 양정×0.65 (자동).

알람밸브 노드 기준으로 평면도(X·Y) + 계통도 라이저 + 기계실(옥상수조) 을 하나의 배관망으로 결합합니다.

통합 배관망 시각화

배관망 통합 후, 캔버스 좌측 상단 버튼으로 평면도 · 등각도 · 아이소메트릭 3D 를 즉시 전환할 수 있습니다. 등각도/3D 는 계통도 라이저(Z)와 기계실/수원 고도를 펼쳐 HAS·solver 화면과 동일한 시점으로 보여줍니다.

출력 좌표 형태

아래 포맷 출력(SDF·KFP·HAS) 파일에 담길 노드 좌표 형태를 선택합니다. 수리계산 결과엔 영향 없음(표시 전용).

등각 출력 시 계통도 라이저(Z)·기계실 고도를 펼치는 강도. 값이 클수록 층이 세로로 더 벌어집니다.

K-Fire Solver 에서 노드가 너무 작/크게 보일 때 .kfp 표시좌표 배율을 조정합니다(기본 1.0, KFP 전용·수리계산 불변). 노드가 작으면 값을 키우고, 그래도 작아지면 1 미만으로 줄여 보세요.

헤드 설치방향

가지배관 헤드를 등각도에서 Z 방향으로 돌출시켜 그립니다. 전역(전체 일괄) · 표시 전용(수리계산 무관).

스케일 무관(도면 대각선 비율). 0 이면 돌출 없음. 평면도엔 표시되지 않고 등각도/3D 에서 보입니다.

포맷 출력

배관망 통합 후 원하는 수리계산 포맷으로 내보냅니다.

아직 배관망 통합이 실행되지 않았습니다.

HAS 불러오기
